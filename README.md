#### W起初用来表示两个V的连写。拉丁语中的 /w/ 发展成为罗曼语中的/v/，因此V不再完全表示日耳曼语系中的/w/。就像在罗曼语系中一样，德语中W丢失了，这就是为什么德语的W表示/v/而不是/w/。在荷兰语中，W是一个approximant（除了带有eeuw的词语，发作/-e:β/）。
